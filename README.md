# Rafael Tonietto — Links Oficiais

Página de links oficiais (estilo link-in-bio) para Rafael Tonietto — Personal Trainer Academy.

## Deploy na Vercel

1. Acesse [vercel.com/new](https://vercel.com/new) e importe o repositório `BryanJohn2901/rafael-tonietto`.
2. Framework Preset: **Other** (site estático HTML).
3. Build Command: deixe em branco.
4. Output Directory: `.` (raiz do projeto).
5. Clique em **Deploy**.

Não é necessário variável de ambiente nem build — o `index.html` na raiz é servido automaticamente.
